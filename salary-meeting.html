③ 建议保留字段（3个）

原讨论删除，建议不删不合并
#字段建议理由
1饭贴顿数 / 饭贴单价/饭贴保留每月浮动不固定，不便于并入固定字段（岗位津贴）
2其他扣款保留手动录入项，如违规罚金和其他扣款项（例如彭旭才）
3固定扣款保留并改名系统自动统计项，改名为 "考勤补贴扣款"
✏️

④⑤⑥ 字段改名 / 移动 / 新增

字段名优化、归属调整、新增员工编号
#调整项类型详情
④ 字段改名 改名 "各类补贴小计" → "考勤补贴小计"
"个税补贴" → "个税补贴（年底统算）"
便于理解和提醒
⑤ 信息补贴 → 岗位津贴 移动 管理人员的"信息补贴"并入岗位津贴作为固定项
原因：在考勤补贴大项里，但不随缺勤而扣款
⑥ 社保补贴移出考勤补贴 移动 从"考勤补贴"大项 → 移入"基础工资架构"大项作为固定项
原因：不随缺勤而扣款
⑩ 新增"员工编号"列 新增 作为员工数据唯一识别号，区别于身份证号码（红线数据）
基础信息和工资架构等固定项会随时间线变化，需按时间线维护更新并建立时间关联
🧮

⑦⑧⑨ 合计公式调整

三个小计/合计项的组成字段变更对比
项合计名称变更前 → 变更后
⑦ 考勤补贴小计
原 = 信息补贴 + 固定饭贴 + 社保补贴 + 个税补贴 + 住房补贴 + 租房补贴（6项）
新 = 信息补贴（不含管理人员）+ 个税补贴 + 租房补贴（3项）
移出：固定饭贴（并入岗位津贴）、社保补贴（→基础工资架构）、住房补贴（并入租房补贴）
信息补贴：管理人员部分并入岗位津贴，非管理人员仍在此项
⑧ 基础工资小计
原 = 基本工资+管理工资+岗位工资+管理津贴+岗位津贴+试用期折减+固定项+危废岗补贴+垃圾房补贴+外围补贴+技能补贴+交通补贴+其他补贴+考核奖+固定代岗+固定加班（16项）
新 = 基本工资+管理工资+岗位工资+管理津贴+岗位津贴+试用期折减+技能补贴+固定代岗+固定加班+社保补贴（10项）
移出：固定项、危废岗补贴、垃圾房补贴、外围补贴、交通补贴、其他补贴、考核奖（均并入岗位津贴）
移入：社保补贴（从考勤补贴大项移入）
⑨ 增项小计
原 = 平日加班+节日加班+动态固定加班费+工龄工资+饭贴+临时代岗费+钟点工+临时工+打卡+全勤奖+特约费+借出补贴+高温补贴+奖金+补发+工程津贴+其他补贴2+固定补贴（18项）
新 = 平日加班+节日加班+工龄工资+饭贴+临时代岗费+钟点工+临时工+打卡+特约费+借出补贴+高温补贴+补发（12项）
移出：动态固定加班费（并入平日加班费）、全勤奖（→奖金审批）、奖金（→奖金审批）、工程津贴（→奖金审批）、其他补贴2（并入岗位津贴）、固定补贴（并入岗位津贴）
